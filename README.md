# Max's Blogg Applikation

## Beskrivning
Denna applikation tillåter användare att skapa, visa, redigera och ta bort blogginlägg. Applikationen är byggd med Node.js, Express.js och EJS som mallmotor.

## Funktioner
1. Skapa inlägg
2. Visa alla inlägg
3. Redigera inlägg
4. Ta bort inlägg
5. Responsiv design
